Image Guide For Choosing File Types (Bootstrap Version)
Below is a reference table that can be used to easily learn about image types. You will find info on different image types, when to use them, tips on naming images, as well as example images.

File Extension When, Where, How? Examples
JPEG Images JPEG files should be used when an image has many different colors. For instance, an image of a sunset, or like this little cute bird. Naming images with their size (length x width) and including an alt value are good practices!
PNG Images PNG files should be used when an image has fewer colors or a large area of the same color. For instance, a logo or simple illustration. Naming images with their size (length x width) and including an alt value are good practices!
GIF Images GIF files should be used when an image has fewer colors or a large area of the same color. For instance, a logo or simple illustration. Naming images with their size (length x width) and including an alt value are good practices!
